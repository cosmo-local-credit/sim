This directory stores the minimal canonical engine-validation gate artifact used
by paper-facing frontier runs when no sibling output-root validation artifact is
available.

The full validation artifacts are generated by:

```bash
./scripts/start_regenbond_batch_tmux.sh validation-full
```

The current committed summary is the 100-run, 260-tick Sarafu engine validation
pass produced with the default validation/frontier own-voucher stable borrowing
probability of 0.70.

Only `engine_validation_summary.csv` is required by the frontier guard. Larger
run summaries, timeseries, figures, and tables remain reproducible outputs and
are intentionally not committed here.
